# Clean Architecture - Use Case Challenge

## Sobre este Projeto

Este repositório contém um desafio de implementação de Clean Architecture com foco em casos de uso (Use Cases) para gerenciamento de produtos, incluindo testes unitários e de integração.

## Pré-requisitos

- Node.js versão 16 ou superior

## Instalação

```bash
npm i
```

## Executando os Testes

```bash
npm test
```

## Desafio de Implementação

O desafio consiste em implementar Use Cases para a entidade Product seguindo os princípios de Clean Architecture:

- **Testes Unitários**: Validar a lógica de negócio isolada
- **Testes de Integração**: Validar a comunicação entre camadas

### Objetivos

- Proteger o domínio da aplicação
- Manter baixo acoplamento entre camadas
- Orientar o código para casos de uso explícitos
- Garantir facilidade em desenvolvimento, deploy, operação e manutenção
